# 2. About Paper

기존의 data augementation 방법은 대부분 heuristic하여 다음과 같이 2가지 문제가 있습니다.

1. Domain에 따라 최적의 augmentation이 다른데, 이를 찾는 과정이 time-consuming함
2. Heuristic한 data augmentation은 종종 'garbage' data를 만듬

TANDA는 dataset이 주어졌을 때, 학습을 통해 해당 dataset에 대한 최적의 augmentation을 찾아줍니다.

Neural architecture search와 매우 비슷한 방법으로 학습이 진행됩니다.

![TANDA Pipeline](../../../../img/210320_TANDA/Figure_1.png)
![Search Space](../../../../img/210320_TANDA/Figure_2.png)

1. Augmentation의 방법 (회전, 평행이동, ...), 정도 (회전으로 예를 들면 10도, 30도, ...), 개수 (몇 개의 augmentation 함수를 사용할 지)를 사람이 heuristic하게 정함
2. Generator가 1번에서 언급된 사람이 정해준 search space에서 Transformation Functions (TF)를 생성
3. 생성된 TF를 이용하여 augmented data를 만듬
4. Discriminator가 input이 augmented 됐는지 아닌지를 classification

즉 generator는 기존 데이터와 비슷한 이미지 분포를 만드는 TF를 생성하게끔 학습합니다.
(TF로 augmented data는 실제 data와 distribution이 비슷하게끔)

![Loss](../../../../img/210320_TANDA/Figure_3.png)

Generator가 inverse한 조합들 (left flip + right flip)을 못배우게끔, loss function에 augmented data와 unaugmented data의 distance를 넣어 학습합니다.
(raw input space와 feature space를 모두 고려한 distance)

Generator model로 2가지 모델을 사용할 수 있습니다.

1. Independent model : mean field model
2. State-based model : LSTM

Generator가 TF를 만드는 과정이 differentiable하지 않기에, 강화학습을 이용하여 학습합니다.

Discriminator는 DCGAN에 사용되는 CNN을 이용합니다.
